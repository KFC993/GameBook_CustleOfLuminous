================================================
Git 超簡潔マニュアル
================================================

■ 毎回やること（これだけ！）
------------------------------------------------
1. フォルダに移動
   cd C:\Github

2. 変更を確認
   git status

3. ファイルを箱詰め
   git add .

4. 封をする
   git commit -m "今日やったこと"

5. GitHubに送る
   git push

------------------------------------------------
基本はこれで終わり！
------------------------------------------------


■ よく使うコマンド
------------------------------------------------
git status        今の状態を見る（困ったらまずこれ）
git add .         全ファイルを箱詰め
git log --oneline これまでの履歴を見る
------------------------------------------------


■ トラブル対処
------------------------------------------------
【add忘れてcommitしちゃった】
→ 何も起きてないので、もう一度addからやり直す

【間違えてaddした】
git reset HEAD ファイル名

【編集を全部なかったことにしたい】
git restore ファイル名
（注意：戻せません）
------------------------------------------------


■ 覚えること3つ
------------------------------------------------
1. 順番：add → commit → push
2. ファイル名にスペース → "で囲む"
3. 困ったら git status
------------------------------------------------


■ 概念メモ
------------------------------------------------
add    = 段ボール箱に詰める
commit = 箱に封をする（記録完成）
push   = GitHubに送る（バックアップ完了）
------------------------------------------------
